Phonebook
-
A phonebook web app, allowing CRUD operations on contacts

Operations User can do
-
     - Add a contact
     - Remove a contact
     - Update a contact
     - Search a contact
To run the application in localhost
-
     - npm install
     - npm start
     - open localhost:3000
It is deployed on
-
https://assignmentphonebook.herokuapp.com/
